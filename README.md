# Game Dev SDL2

SDL2 for Game development. Beginner to Advanced.

## Requirement

- vcpkg (C++ Package Manager)
- sdl2  `(vcpkg install sdl2)`
- cmake

## Compile
This repository

```
mkdir build
cd build
cmake .. "-DCMAKE_TOOLCHAIN_FILE=E:/sdk/vcpkg/scripts/buildsystems/vcpkg.cmake"
cmake --build .
```
